# Zuri Market Frontend

This is the React frontend for the Zuri Market project.

## Setup

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

> The frontend expects the backend to be available at `http://localhost:5000` during development. Vite is configured to proxy `/api` requests to the backend.

Run frontend tests:

```bash
npm run test
```

Build for production:

```bash
npm run build
```
